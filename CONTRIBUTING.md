# Contributing to ParallaxKey

Thank you for your interest in contributing to ParallaxKey! We welcome contributions from the community to help make web development more secure.

## How to Contribute

1. **Fork the Repository**: Create a fork of this repo on GitHub.
2. **Clone forking**:
   ```bash
   git clone https://github.com/your-username/ParallaxKey.git
   ```
3. **Create a Branch**: 
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make Changes**: Implement your feature or bug fix.
5. **Run Tests**: Ensure your changes don't break existing functionality.
   ```bash
   npm run build
   ```
6. **Commit & Push**:
   ```bash
   git commit -m "feat: add amazing feature"
   git push origin feature/your-feature-name
   ```
7. **Open a Pull Request**: Submit your changes for review.

## Code Standards
- Use TypeScript for all logic.
- Follow the existing project structure in `src/`.
- Ensure new scanners extend the `BaseScanner` class.
- Use Tailwind CSS for styling.

## Reporting Issues
Please use the GitHub Issues tab to report bugs or suggest new features. Include:
- A clear description of the issue.
- Steps to reproduce (for bugs).
- Your environment (browser version, OS).
